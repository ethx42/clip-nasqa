**Step 2: Create `packages/frontend/src/lib/safe-action.ts`**

Export a generic wrapper function:

```typescript
import type { ActionResult } from "./action-result";

/**
 * Wraps a server action call to catch network-level fetch failures
 * (e.g., ERR_INTERNET_DISCONNECTED) and convert them to ActionResult.
 */
export async function safeAction<T>(
  action: Promise<ActionResult<T>>,
  networkErrorMessage: string,
): Promise<ActionResult<T>> {
  try {
    return await action;
  } catch {
    return { success: false, error: networkErrorMessage };
  }
}
```

The second parameter `networkErrorMessage` is the pre-translated string. Callers pass in the translated message from their component context (where `useTranslations` is available). This avoids importing `useTranslations` in a non-component utility.
