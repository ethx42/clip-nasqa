2. Create `packages/frontend/src/actions/session.ts`:
   - Add `'use server'` directive at top.
   - Import `redirect` from `next/navigation`, `createHash`, `randomUUID` from `node:crypto`, `PutCommand` from `@aws-sdk/lib-dynamodb`, `docClient` and `tableName` from `@/lib/dynamo`.
   - Export `async function createSession(formData: FormData)`:
     a. Extract and validate title: `(formData.get('title') as string)?.trim().slice(0, 50)`. If empty, throw error.
     b. Dynamic import `generateSlug` from `random-word-slugs`.
     c. Retry loop (max 3 attempts) for slug collision:
     - Generate slug: `generateSlug(2, { format: 'kebab' })`.
     - Generate rawSecret: `randomUUID()`.
     - Hash secret: `createHash('sha256').update(rawSecret).digest('hex')`.
     - Calculate `now = Math.floor(Date.now() / 1000)`.
     - PutCommand with Item: `{ PK: \`SESSION#\${slug}\`, SK: \`SESSION#\${slug}\`, slug, title, hostSecretHash: hashedSecret, isActive: true, createdAt: now, TTL: now + 86400 }`.
     - ConditionExpression: `'attribute_not_exists(PK)'`.
     - On success: `redirect(\`/session/\${slug}/success?raw=\${rawSecret}\`)`. Note: the redirect will include the current locale prefix automatically via next-intl.
     - On `ConditionalCheckFailedException`: increment attempts, continue loop.
     - On other errors: re-throw.
       d. After 3 failed attempts: throw `new Error('Failed to generate unique slug')`.
   - IMPORTANT: `redirect()` in Next.js throws a special error internally. Do NOT catch redirect errors. Structure the try/catch so that `redirect()` is called OUTSIDE the try block, or check for the NEXT_REDIRECT error type before rethrowing.

3. Create `packages/frontend/src/lib/session.ts`:
   - Import `GetCommand` from `@aws-sdk/lib-dynamodb`, `docClient` and `tableName` from `@/lib/dynamo`.
   - Export `async function getSession(slug: string)`:
     a. Send GetCommand with `Key: { PK: \`SESSION#\${slug}\`, SK: \`SESSION#\${slug}\` }`.
b. If no Item, return null.
c. TTL filter (SESS-07): if `item.TTL < Math.floor(Date.now() / 1000)`, return null.
d. Return a clean session object: `{ slug: item.slug, title: item.title, isActive: item.isActive, createdAt: item.createdAt, expiresAt: item.TTL }`. Do NOT expose PK, SK, or hostSecretHash.
   - Export the `Session` TypeScript type matching that return shape.
