   - `handleBanParticipant(args: BanParticipantArgs)`: Verify host secret. PutCommand or UpdateCommand on `SESSION#slug / BAN#fingerprint` with `isBanned = true`, `bannedPostCount = if_not_exists(0)`, `TTL = now + 86400`. Broadcast PARTICIPANT_BANNED event with `{ fingerprint }` payload.

After update, compute `shouldHide = downvoteCount / (upvoteCount + downvoteCount) >= 0.5` from ReturnValues. If shouldHide differs from current isHidden, do a second UpdateCommand to SET isHidden. Broadcast QUESTION_UPDATED with updated fields.
