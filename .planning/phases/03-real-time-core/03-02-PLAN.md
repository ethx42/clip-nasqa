**1. Create packages/frontend/src/lib/detect-language.ts:**

Client-safe module (no server imports). Export `detectLanguage(code: string): string` function that uses simple regex heuristics per research Pattern 7:

```typescript
export function detectLanguage(code: string): string {
  const trimmed = code.trim();
  if (!trimmed) return 'text';
  // Check for common language patterns
  if (/^\s*(import\s|from\s|export\s|const\s|let\s|var\s|function\s|class\s|interface\s|type\s|=>)/.test(trimmed) && /[{}]/.test(trimmed)) return 'typescript';
  if (/^\s*(def\s|import\s|from\s|class\s|if\s+__name__|print\()/.test(trimmed)) return 'python';
  if (/^\s*(func\s|package\s|import\s*\(|:=\s)/.test(trimmed)) return 'go';
  if (/^\s*(public\s+class|private\s|System\.out)/.test(trimmed)) return 'java';
  if (/^\s*#include\s|int\s+main\s*\(|std::/.test(trimmed)) return 'cpp';
  if (/<\/?[a-z][\s\S]*>/i.test(trimmed) && !/^\s*\{/.test(trimmed)) return 'html';
  if (/^\s*\{[\s\S]*"[\w]+"[\s\S]*:/.test(trimmed)) return 'json';
  if (/^\s*(SELECT|INSERT|UPDATE|DELETE|CREATE|ALTER|DROP)\b/i.test(trimmed)) return 'sql';
  if (/^\s*[\w-]+\s*:/.test(trimmed) && !/{/.test(trimmed)) return 'yaml';
  if (/^\s*(fn\s|let\s+mut|impl\s|use\s+std)/.test(trimmed)) return 'rust';
  if (/[{;]/.test(trimmed) && /\w+\s*\(/.test(trimmed)) return 'javascript';
  return 'text';
}
```

Also export a `SUPPORTED_LANGUAGES` array of `{ value: string, label: string }` for the language selector chip: typescript, javascript, python, go, java, rust, cpp, html, css, sql, json, yaml, bash, text.

**2. Create packages/frontend/src/actions/snippet.ts:**

`'use server'` directive. Two Server Actions:

- `renderHighlight(code: string, lang: string): Promise<string>` — calls `codeToHtml(code, { lang: lang as BundledLanguage, themes: { light: 'github-light', dark: 'github-dark' } })` and returns the HTML string. This powers the host live preview without putting Shiki in the client bundle.

- `pushSnippetAction(formData: { sessionSlug: string, hostSecretHash: string, content: string, type: string, language?: string }): Promise<{ ok: boolean, error?: string }>` — calls the AppSync pushSnippet mutation via the appsyncClient. Import `appsyncClient` from `@/lib/appsync-client` and `PUSH_SNIPPET` from `@/lib/graphql/mutations`. Execute `appsyncClient.graphql({ query: PUSH_SNIPPET, variables: formData })`. Return `{ ok: true }` on success, `{ ok: false, error: message }` on failure.
